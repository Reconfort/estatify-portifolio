# @estatify/branding

Estatify **platform** branding only — logos, brand marks, favicons, OpenGraph /
social images, official illustrations.

Do **not** put here: application-specific assets (those live in `apps/*/public`),
reusable non-branded media (`@estatify/media`), or tenant uploads (object storage).

Import via alias only: `import { EstatifyLogo } from "@estatify/branding"`.
