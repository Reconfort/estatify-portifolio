# @estatify/media

Reusable, **non-branded** media shared across apps — default avatar, property
placeholder, empty-state and loading graphics.

Do **not** put here: branding (`@estatify/branding`), app-specific assets
(`apps/*/public`), or tenant uploads (object storage — never in Git).

Import via alias only: `import { PropertyPlaceholder } from "@estatify/media"`.
